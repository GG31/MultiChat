# MultiChat

To install a dependency named XYZ, you have to type on the console :
```
[sudo] npm install XYZ [-g]
```

Dependencies to install :

express
mongodb
socket.io
delivery
body-parser


To access a room : 
http://localhost:2013/room/roomname
