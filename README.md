# MultiChat
use **NPM** to install all the dependencies
```
npm install
```

Install express
```
sudo npm install express -g
```

Install mongodb
```
sudo apt-get install mongodb
```
Install delivery
```
sudo npm install delivery
```
Pour accéder à une room : 
http://localhost:2013/room/roomname
